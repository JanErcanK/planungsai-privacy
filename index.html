Datenschutzerklärung für PlanungsAI

Stand: Januar 2025

1. Verantwortlicher

Diese App wird von einer Privatperson betrieben. Bei Fragen zum Datenschutz kannst du dich jederzeit melden unter:
E-Mail: [DEINE E-MAIL]

2. Erhebung und Verarbeitung von Daten

PlanungsAI speichert ausschließlich die vom Nutzer selbst eingegebenen Informationen, wie z. B. Termine, Beschreibungen oder Kontextdaten für die KI-Planungsfunktion.

Es werden keine personenbezogenen Daten wie Name, Telefonnummer, Adresse oder Zahlungsinformationen automatisch erfasst.

3. Verwendete Dienste

Firebase Firestore (Google)

Die App nutzt Google Firebase Firestore zur Speicherung der Event-Daten. Dabei werden Daten sicher übertragen und verschlüsselt gespeichert.

Cloud Function / KI-API

Zur Verarbeitung deiner Eingaben wird eine von dir bereitgestellte Cloud Function genutzt, die Anfragen an ein KI-Modell weiterleitet (z. B. Anthropic Claude). Es werden nur die Inhalte gesendet, die du explizit eingibst.

4. Weitergabe von Daten

Es erfolgt keine Weitergabe oder Verkauf von Nutzerdaten an Dritte.

Daten werden ausschließlich zur Funktion der App verarbeitet.

5. Rechte der Nutzer

Du hast jederzeit das Recht auf:

  • Auskunft über gespeicherte Daten
  • Löschung deiner Daten
  • Korrektur fehlerhafter Einträge

Eine vollständige Löschung kann jederzeit über eine Anfrage per E-Mail erfolgen.

6. Datensicherheit

Alle Daten werden verschlüsselt übertragen (HTTPS). Die Speicherung in Firestore erfolgt verschlüsselt gemäß den Standards von Google Cloud.

7. Änderungen der Datenschutzerklärung

Diese Richtlinie kann in Zukunft angepasst werden, falls neue Funktionen hinzukommen.
